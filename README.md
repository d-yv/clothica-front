
#
## Clothica-shop

##
Clothica-shop — це простий і зручний онлайн-магазин одягу. На сайті можна переглядати одяг за категоріями, фільтрувати за розміром, ціною та типом товарів. Кожен товар має фото, опис, розмірну сітку та рейтинг. Є зручний кошик, який відкривається збоку — там можна змінювати кількість товарів або видаляти їх. Оформлення замовлення проходить через просту форму з контактними даними. Сайт виглядає мінімалістично, працює швидко і підлаштовується під телефон, планшет та комп’ютер, щоб користувачу було максимально легко робити покупки.

##
Проект реалізовано з використанням Next.js, TS, CSS. Бібліотекі axios, formik, swiper, yup, zustand.
